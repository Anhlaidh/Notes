# Note
## 基本概念
### 进程 线程 纤程(quasar)
- Java线程,详见Java笔记
    - 补充,可以有lambda表达式写法
    - 三种启动线程方式
        1. Thread
        2. Runnable
        3. Executors.newCachedThread(本质也是调用上两个)
    - sleep ,睡眠多少秒,其他线程可以运行
    - yield,加入等待队列,有可能立马还是自己
    - join , 调用另一个线程,保证顺序
    - interrupt , 抛出异常,处理catch
- 锁
    - synchronized
