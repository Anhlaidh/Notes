## 面向对象编程的原则

- 封装
    - 每个x,y,dir都应只属于一个坦克,封装出来,避免代码冗余
        - 如果封装完直接tank.x,就相当于封装再拆封,破坏了封装性
            - 画tank把画笔给tank,让tank自己画自己
- 继承
- 多态
- 持有对方的引用
### 思路
