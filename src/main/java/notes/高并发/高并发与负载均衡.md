## Keepalived
- 问题: lvs和rs会挂
- 主备 主主
- 代替人 自动运维 解决单点故障 实现HA
1. 监控自己服务
2. Master通告自己还活着, Backuo监听Master状态,Master挂了, 一堆Backup推举出一个新的Master
3. 配置:vip 添加ipvs keepalived有配置文件
4. 对后端server做健康检查
- keepalived是一个通用的工具,主要作为HA实现
nginx:可以作为公司的负载均衡来用,nginx陈为了单点故障,也可以用keepalived来解决
